PleaseDoNotAccess.io
Hello? Welcome to PleaseDoNotAccess.io. This project was created to enhance people's understanding about web accessibility. We specifically aim to provide you with an opportunity to consider web accessibility from the perspective of visually impaired individuals.

Before you press the start button below to experience examples of severely compromised web accessibility, we'd like to explain a few helpful things.

1. By pressing the menu bar at the top, you can switch to views designed for individuals with different visual conditions, such as color blindness, red-green color blindness, and low vision. Pressing the front button will return you to the initial screen. We recommend choosing one of the options before starting the experience for a more realistic encounter.

2. Additionally, we encourage you to navigate through the pages using only the keyboard, such as the tab key and arrow keys, instead of using the mouse. This is to make the experience more similar to the actual conditions faced by visually impaired individuals.

Are you ready? Then let's press the start button!
